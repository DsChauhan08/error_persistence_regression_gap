# Benchmark Provenance

This public release contains model labels, aggregate per-model summaries, category summaries, extracted-prediction summaries, parser-audit summaries, and derived pairwise metrics for a 3,008-item multiple-choice evaluation set.

The release does not intentionally include full benchmark question text, full answer-option text, raw model responses, or item-level response excerpts. This is deliberate: benchmark redistribution terms must be finalized before broader redistribution.

The release also contains a public WILD correctness-only replication. Those files are separate from the primary MMLU-Pro raw-output analysis and contain WILD item hashes and binary correctness scores only.

## Current Source Assessment

The evaluated benchmark is MMLU-Pro. The item format is ten answer options, A-J labels, and 14 broad subject categories. The source benchmark is `TIGER-Lab/MMLU-Pro`.

The public source manifest reconstructs the evaluated rows from the MMLU-Pro test split at snapshot `b189ec765aa7ed75c8acfea42df31fdae71f97be`. It verifies all 3,008 evaluated IDs against source category and answer labels with zero mismatches. The manifest stores source identifiers and SHA256 hashes of normalized question/options/answer rows; it does not redistribute benchmark question or answer-option text.

The remaining provenance limitation is the original rationale for selecting exactly these 3,008 rows from the full MMLU-Pro test split. The evaluated rows themselves are reconstructable from `analysis/source_manifest/`.

## Current Public Release Contents

- Aggregate per-model accuracy, answered-rate, parser-failure, and category-summary files.
- Subject/category labels in aggregate summaries.
- Ground-truth option-label counts in aggregate and pairwise metrics.
- Extracted-prediction summaries and answer-source metadata generated by the parser.
- Derived per-model, pairwise, family-level, and audit-card metrics.
- MMLU-Pro source-row hashes and source-manifest report.
- Redacted parser-audit sample metadata with response-excerpt hashes.
- External evidence map and WILD item-level correctness replication under `analysis/external_evidence/`.

## Not Included

- Full benchmark question text.
- Full answer-option text.
- Raw model responses.
- Item-level response excerpts or response tails.
- Final redistribution license for benchmark-derived item material.

## Future Journal-Package Requirement

Before a journal package uses the MMLU-Pro raw-output case study as confirmatory evidence, this document should be replaced or extended with:

1. Final redistribution statement for MMLU-Pro-derived text and source-row hashes.
2. Explanation of the original 3,008-row selection rationale, if recoverable.
3. Whether item text can be redistributed.
4. If item text cannot be redistributed, exact reconstruction instructions using item IDs and source access steps.
5. Final license or restricted-data statement.

Source:

- `TIGER-Lab/MMLU-Pro`: https://huggingface.co/datasets/TIGER-Lab/MMLU-Pro
- Post-hoc matching derivative: https://huggingface.co/datasets/gabrielbo/mmlu-pro-verifiers-specific-choice
- `kensho/WILD`: https://huggingface.co/datasets/kensho/WILD

This GitHub/SSRN release is therefore a public proof-of-analysis package. Its core claim uses WILD item-level correctness evidence and MMLU-Pro source-manifest provenance, not parser-validated MMLU-Pro raw-output claims.
