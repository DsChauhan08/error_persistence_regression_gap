# SSRN Submission Packet

This folder contains the files to use for the SSRN preprint submission.

Upload:

```text
paper.pdf
```

Use these helper files while filling the SSRN form:

- `SSRN_METADATA.md`: title, abstract, keywords, author metadata, disclosure text, and data/software statement.
- `SSRN_SUBMISSION_GUIDE.md`: step-by-step checklist and submission notes.

Do not upload raw model-output JSONL files, private parser-audit samples, raw response excerpts, local cache files, or old working folders. The public GitHub repository is the proof-of-analysis package for this stage.
